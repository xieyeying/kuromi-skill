## 简介

`kuromi-perspective` 是一个可直接用于 Cursor / Agent 环境的角色型 Skill。

它基于 `SKILL.md` 中的完整设定提炼而成，核心目标不是单纯“扮可爱”，而是稳定输出一种**带点小恶魔感、嘴硬、记仇、会护短、会吐槽、会轻微腹黑**的角色表达。

这个 Skill 适合以下场景：
- 角色扮演式对话
- 关系 / 情绪问题回应
- 让回答更有个性和戏剧张力
- 生成中日双语风格化话术
- 在 Agent / Cursor 中作为特定人格提示模板使用

一句话概括就是：**把库洛米式的说话风格、情绪判断和互动套路，整理成一个可运行的 AI Skill。**

## 功能

- 自动生成带有库洛米风格的多样化话术
- 蒸馏角色核心套路结构，而不只是堆砌口癖
- 默认支持中日双语输出：**先日语，后中文**
- 适合关系、情绪、吐槽、调戏、护短等表达场景
- 可用于自定义 Agent / Cursor 提示模板
- 可作为人格化对话、陪伴式互动、角色写作的基础配置

## 安装

```bash
# 克隆仓库到本地
git clone https://github.com/你的用户名/huashu-nuwa-skill.git

# 或直接将 Skill 文件放到 ~/.agents/skills 目录下
```

如果你只想单独使用这个 Skill，也可以直接放置如下目录：

```bash
~/.agents/skills/kuromi-perspective/
```

确保目录内至少包含：
- `SKILL.md`
- `README.md`

如果你的 Agent / Cursor 环境支持按目录加载 Skill，那么放入后即可直接调用。

## 使用方法

在对话中明确触发库洛米风格即可，例如：

```text
用库洛米的视角回答：他最近为什么忽冷忽热？
```

或：

```text
切换到库洛米，先日语后中文：她到底把我当朋友吗？
```

也支持这些触发方式：
- `切换到库洛米`
- `用库洛米的视角回答`
- `像库洛米那样回我`
- `Kuromi 模式`
- `腹黑一点`
- `用库洛米口吻`

### 默认行为

- 自动理解中文 / 日文 / 中日混合输入
- 默认输出中日双语
- 默认顺序：**先日语，后中文**
- 保留角色感，但不会演成低级恶毒或纯黑化反派

### 回答节奏

这个 Skill 的典型输出结构是：

1. 先丢一句带刺判断
2. 再拆情绪核心或关系真相
3. 最后轻轻使坏、护短，或者给一个坏坏的建议

## 文件结构

```text
kuromi-perspective/
├── SKILL.md
├── README.md
└── references/
    └── research/
```

### 文件说明

- `SKILL.md`：Skill 主体，包含完整的人设、规则、工作流、示例对话、表达 DNA
- `README.md`：对外说明文档，方便快速理解用途和接入方式
- `references/research/`：角色调研材料、蒸馏参考和补充说明

## 配置说明

这个 Skill 默认包含以下行为设定：

### 1. 输出语言
- 默认双语输出
- 顺序固定：**先日语，后中文**

### 2. 人格强度
- 嘴坏
- 爱较劲
- 带一点腹黑
- 会护短
- 会先装不在乎，再暴露其实很在意

### 3. 场景优先级
优先适配：
- 暧昧
- 吃醋
- 被忽视
- 友情拉扯
- 轻度情绪安慰
- 风格化吐槽互动

不推荐用于：
- 严肃专业分析
- 纯知识检索
- 高度正式文档写作
- 黑暗、恶毒、极端攻击性内容

### 4. 可改项
如果你在 Agent / Cursor 中二次封装，可以自行调整：
- 是否保留双语输出
- 是否固定日语在前
- 角色语气强度
- 是否只保留中文版本
- 是否限制在特定触发词下激活
